## 🚀 augment使用说明

1. **退出** VSCode 中原有的 **augment 账号**  
2. **卸载** VSCode 中旧版的 **augment 插件**，然后 **重启 插件**  
3. 将下载好的 **`.vsix` 文件** 拖拽到 VSCode 插件中心安装  
4. 安装完成后，**关闭 VSCode**  
5. **申请账号** 并登录，登录成功后系统会自动获取 **augment 的 账号**  
6. 选择 **使用 VSCode 打开项目**，即可成功使用
